tmux cheatsheet

Prefix is C-a — hold Ctrl, tap a, release, then press the next key.

★ = custom binding from your tmux.conf. Everything else is a tmux default.

Sessions — from the shell

tmuxstart a new unnamed session
★ tmmattach/create the main session
★ tmaattach to the most-recent session
★ tmat workattach to session work
★ tms worknew session named work
★ tmllist sessions (tmux ls)
★ tmk workkill session work

Open a terminal & zsh auto-lands you in tmux: first one gets main, reopen a closed one reattaches main, extra terminals get their own independent session (no mirroring). Bare shell: NO_AUTO_TMUX=1 zsh.

Sessions — inside tmux

★ prefix Tsesh popup — fuzzy-jump to any session/dir/project
prefix ddetach (session keeps running)
prefix svisual session/window tree
prefix $rename current session
prefix ( / )previous / next session

Windows (like tabs)

★ prefix cnew window (in current pane's dir)
prefix ,rename window
prefix 1…9jump to window by number
prefix n / pnext / previous window
prefix wlist windows
prefix &kill window

Panes (splits)

★ prefix |split side-by-side (vertical border)
★ prefix -split stacked (top/bottom)
★ C-h C-j C-k C-lno prefix — move between panes & into Neovim splits
★ prefix H/J/K/Lresize pane (hold & repeat)
prefix zzoom pane fullscreen (toggle)
prefix xkill pane
prefix spacecycle pane layouts
prefix { / }swap pane position
prefix !break pane out into its own window

Copy / scroll mode (vi-style)

prefix [enter copy mode (scroll with k/j, C-u/C-d, wheel)
★ vstart selection
★ ycopy selection → system clipboard, exit
/ textsearch in scrollback
qquit copy mode

Mouse is on: click panes, drag borders, scroll with the wheel.

Nested / SSH tmux

C-atalks to the outer (local) tmux
★ C-a C-atalks to the inner (remote) tmux — send-prefix

e.g. new window on the remote: C-a C-a c

Plugins (TPM)

prefix Iinstall plugins listed in config (capital i)
prefix Uupdate plugins
prefix alt-uremove plugins no longer listed

Handy meta

★ prefix rreload ~/.config/tmux/tmux.conf
prefix ?list every keybinding (live cheatsheet)
prefix :command prompt
prefix tbig clock

A typical first workflow

1.tmm — land in the main session (zsh auto-attaches here too)
2.prefix | to split, C-l/C-h to hop panes
3.prefix c for a 2nd window, prefix 1/2 to switch
4.prefix d to detach → later tma or prefix T
5.edit the config, then prefix r to reload

Memorize first: prefix z (zoom) and prefix T (sesh). Forgot something? prefix ?.
